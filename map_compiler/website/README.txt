Map Compiler WebSite:
This site in the enty point for the map creation of the Autonomous Navigation Project:
Steps For Create a map:
1.- Upload a Initial Aerial Photos to process over ODM command line.
2.- **Ortographic Pipeline** will create for you GeoTif files as input for the next
3.- **Map Tools** will identify lanes, landscape, soil, vegetation, obstacles
4.- Create tiles and segmentate created features and localize it
    Create and derive Links/lanes/obtacles and roads from previous steps.
5.- Create DataBase and phisical model(Protobuff model) of the map
6.- Upload the created DataBase for user manipulation
